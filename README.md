# opencv.dockerfile
pure opencv container image without Python bindings

# Quick Start

make all

# Prerequisite

### Python

```
sudo ./install-packages
```

edit build file to meet your needs

```
sudo ./build-python3
```

# Build darkflow

```
sudo ./build-darkflow
```

# Build mkl

```
sudo ./build-mkl
```

# Build Caffe

```
sudo ./build-caffe
```
